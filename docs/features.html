Features

Everything you need to edit image-based slides

01

On-Device OCR

SlideTouch uses Apple's Vision framework to detect and recognize text in image-based slides. Text regions are identified with precise bounding boxes, and you can click any detected text to start editing. Supports English, Korean, and Japanese.

Screenshot coming soon
02

AI-Powered Inpainting

When you edit text, the original characters need to be erased first. SlideTouch uses the LaMa inpainting model running entirely on your Mac's Neural Engine to restore the background behind removed text. Works on gradients, patterns, and photographs.

Screenshot coming soon
03

Non-Destructive Layer Editing

Every edit in SlideTouch is a layer on top of the original image. Nothing is permanently changed until you export. You can toggle, reorder, or delete any edit layer at any time, with full undo/redo support.

Screenshot coming soon
04

Object Selection & Removal

Select and remove objects from slides using SAM2 segmentation. Draw a stroke over any element, and the AI generates a precise mask. The object is removed and the background is seamlessly restored.

Screenshot coming soon
05

Shape Drawing

Add annotations with rectangles, ellipses, arrows, and freeform curves. Customize stroke color, fill color, and line width. Perfect for highlighting important areas or adding visual emphasis.

Screenshot coming soon
06

Rich Text Editing

Full control over typography: choose fonts, adjust sizes, set colors, and align text. Apply different styles to individual text runs within a single text box for maximum flexibility.

Screenshot coming soon
07

Patch-Based Processing

SlideTouch never sends full 4K+ images through ML models. Instead, it crops to the affected region with context padding, processes just that patch, and composites it back. This preserves quality and dramatically improves performance.

Screenshot coming soon
08

Flexible Export

Export your edited slides as a new PDF document, individual PNG/JPEG images, a video slideshow, or an animated GIF. The original PDF file is never modified.
