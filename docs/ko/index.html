기능 살펴보기 릴리즈 노트

우리가 해결하는 문제

😤

이미지 슬라이드는 편집이 불가능

NotebookLM, Canva 등에서 만든 슬라이드는 아름답지만, 전체가 이미지로 평탄화되어 있습니다. 오타 하나를 고치려면 전체 덱을 다시 생성해야 합니다.

✨

SlideTouch가 편집 가능하게 만듭니다

온디바이스 OCR로 텍스트를 감지하고, Neural Engine 인페인팅으로 깔끔하게 지운 뒤, 정밀한 렌더링으로 새 텍스트를 배치합니다. 슬라이드의 어떤 픽셀이든 편집할 수 있습니다.

핵심 기능

🔍

온디바이스 텍스트 감지

Apple Vision 프레임워크가 이미지 슬라이드에서 텍스트를 감지합니다. Mac에서 완전히 실행됩니다.

🎨

매끄러운 텍스트 교체

원본 텍스트가 지워지고 Neural Engine이 배경을 복원한 후, 새 텍스트가 그 자리에 렌더링됩니다.

🎯

오브젝트 선택 및 이동

스트로크를 그려 요소를 선택하세요. 이동, 위치 변경, 제거 가능. 배경이 매끄럽게 채워집니다.

📐

비파괴 편집

모든 편집은 오버레이 레이어. 무제한 실행 취소/다시 실행. 내보내기 전까지 변경 없음.

✏️

리치 텍스트 및 도형

글꼴, 크기, 색상, 정렬 완벽 제어. 사각형, 타원, 화살표, 자유곡선도 지원합니다.

📤

다중 포맷 내보내기

PDF, PNG, JPEG, 비디오 슬라이드쇼, 애니메이션 GIF로 내보내기. 원본 파일은 수정되지 않습니다.

🔒 100% 온디바이스 처리. 파일이 Mac 밖으로 나가지 않습니다.

작동 방식

1

PDF 열기

이미지 기반 PDF를 SlideTouch에 드롭하세요. 페이지가 원본 해상도로 렌더링됩니다.

2

편집

텍스트를 클릭하면 편집이 시작됩니다. 원본은 인페인팅으로 지워지고 새 텍스트가 제자리에 렌더링됩니다.

3

내보내기

새 PDF, 이미지, 또는 비디오로 저장하세요. 원본 파일은 절대 수정되지 않습니다.
