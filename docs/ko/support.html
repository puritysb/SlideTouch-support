SlideTouch 사용 중 문제가 있다면 아래 내용을 확인해 주세요.

시스템 요구사항

  • macOS 15.1+ (Sequoia 이상)
  • Apple Silicon (M1, M2, M3, M4) — Intel Mac은 지원하지 않습니다
  • 8GB+ RAM 권장

자주 묻는 질문

왜 Apple Silicon이 필요한가요?

SlideTouch는 Apple Silicon 칩의 Neural Engine에 의존하는 Core ML 모델을 사용합니다. Intel Mac에는 이 하드웨어가 없어서, CPU만으로는 추론 속도가 실용적이지 않습니다.

일반 (텍스트 기반) PDF도 편집할 수 있나요?

SlideTouch는 텍스트가 편집 가능한 객체가 아닌 픽셀로 존재하는 이미지 기반 PDF를 위해 설계되었습니다. 텍스트 기반 PDF에는 일반 PDF 편집기가 더 적합합니다.

원본 PDF가 수정되나요?

절대 수정되지 않습니다. SlideTouch는 엄격한 비파괴 편집 원칙을 따릅니다. 모든 편집은 오버레이 레이어로 저장됩니다. 내보내기 시 새 파일이 생성되며, 원본은 그대로 유지됩니다.

OCR은 어떤 언어를 지원하나요?

현재 영어, 한국어, 일본어를 지원합니다. 언어 지원은 Apple Vision 프레임워크의 인식 기능에 따릅니다.

인터넷 연결이 필요한가요?

아닙니다. 모든 처리는 기기에서 완전히 이루어집니다. SlideTouch는 네트워크 기능이 없으며, 파일을 어디에도 업로드하지 않습니다.

인페인팅 결과가 흐릿하거나 아티팩트가 있어요

더 작은 편집 영역을 사용해 보세요. 인페인팅 모델은 작은~중간 크기의 패치에서 가장 잘 작동합니다. 매우 넓은 영역은 여러 번 나눠서 편집하는 것이 좋습니다.

앱이 메모리를 많이 사용해요

SlideTouch는 ML 모델을 필요할 때 로드하고 30초간 사용하지 않으면 해제합니다. 매우 큰 PDF(100+ 페이지)의 경우, 활발히 편집하지 않는 페이지를 닫으면 메모리 사용량이 줄어듭니다.

버그 신고

버그를 신고하는 가장 쉬운 방법은 GitHub Issues입니다.

  • Issues: github.com/puritysb/SlideTouch-support/issues

버그 신고 시 포함할 내용

  • 앱 버전 (SlideTouch > SlideTouch에 관하여)
  • macOS 버전 및 칩 모델 (예: macOS 15.2, M2 Pro)
  • 사용 중이던 기능 (OCR, 인페인팅, 내보내기 등)
  • 문제 재현 방법
  • 가능하면 스크린샷

개인정보 보호

SlideTouch는 모든 것을 기기에서 처리하며 네트워크 접근이 없습니다. 자세한 내용은 개인정보처리방침을 참고하세요.
