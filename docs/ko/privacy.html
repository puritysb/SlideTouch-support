개인정보처리방침

최종 업데이트: 2026-02-08

요약

  • SlideTouch는 모든 것을 100% 기기에서 처리합니다.
  • 앱은 네트워크 접근이 없으며, 파일을 어디에도 업로드하지 않습니다.
  • 광고, 추적, 분석 SDK가 포함되어 있지 않습니다.

접근하는 정보

SlideTouch는 시스템 파일 선택기나 드래그 앤 드롭을 통해 사용자가 명시적으로 여는 파일만 접근합니다:

  • 편집하려고 선택한 PDF 파일 및 이미지
  • 저장을 위해 선택한 내보내기 대상 폴더

SlideTouch는 사진 라이브러리, 연락처, 캘린더 또는 기타 시스템 데이터에 접근하지 않습니다.

온디바이스 처리

모든 처리가 Mac에서 로컬로 실행됩니다:

  • OCR: Apple Vision 프레임워크 (VNRecognizeTextRequest)
  • 인페인팅: Neural Engine에서 실행되는 LaMa FP16 Core ML 모델
  • 세그멘테이션: Neural Engine에서 실행되는 SAM2 Tiny Core ML 모델
  • 렌더링: Core Graphics 및 Core Image

클라우드 API 없음, 원격 추론 없음, 어떤 종류의 서버 통신도 없습니다.

네트워크 접근

SlideTouch는 네트워크 권한이 없습니다. 앱은 샌드박스 처리되어 물리적으로 네트워크 요청을 할 수 없습니다. 문서는 완전히 기기에 남아 있습니다.

ML 모델

머신러닝 모델(LaMa, SAM2)은 설치 시 앱과 함께 번들됩니다. 인터넷에서 다운로드되지 않습니다. 모델 추론은 전적으로 Mac의 Neural Engine에서 실행됩니다.

데이터 저장

SlideTouch는 앱 환경설정과 최근 파일 참조만 샌드박스 컨테이너에 저장합니다. 내보내기 기능을 통해 명시적으로 저장하는 것 외에는 사용자 콘텐츠가 캐시되거나 저장되지 않습니다.

데이터 삭제

SlideTouch를 삭제하면 모든 앱 데이터가 함께 삭제됩니다. 원본 PDF 파일은 SlideTouch에 의해 수정되지 않으며, 저장한 위치에 그대로 남아 있습니다.

문의
