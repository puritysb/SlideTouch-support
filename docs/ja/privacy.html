プライバシーポリシー

最終更新日: 2026-02-08

概要

  • SlideTouchはすべてを100%デバイス上で処理します。
  • アプリにはネットワークアクセスがなく、ファイルをどこにもアップロードしません。
  • 広告、トラッキング、分析SDKは含まれていません。

アクセスする情報

SlideTouchは、システムのファイル選択ダイアログまたはドラッグ＆ドロップを通じてユーザーが明示的に開いたファイルのみにアクセスします：

  • 編集用に選択したPDFファイルおよび画像
  • 保存用に選択した書き出し先フォルダ

SlideTouchは写真ライブラリ、連絡先、カレンダー、その他のシステムデータにはアクセスしません。

オンデバイス処理

すべての処理がMac上でローカルに実行されます：

  • OCR: Apple Visionフレームワーク（VNRecognizeTextRequest）
  • インペインティング: Neural Engineで動作するLaMa FP16 Core MLモデル
  • セグメンテーション: Neural Engineで動作するSAM2 Tiny Core MLモデル
  • レンダリング: Core GraphicsおよびCore Image

クラウドAPIなし、リモート推論なし、いかなる種類のサーバー通信もありません。

ネットワークアクセス

SlideTouchにはネットワーク権限がありません。アプリはサンドボックス化されており、物理的にネットワークリクエストを送信できません。ドキュメントは完全にマシン上に留まります。

MLモデル

機械学習モデル（LaMa、SAM2）はインストール時にアプリにバンドルされます。インターネットからダウンロードされることはありません。モデルの推論はすべてMacのNeural Engineで実行されます。

データ保存

SlideTouchはアプリの設定と最近のファイル参照のみをサンドボックスコンテナに保存します。書き出し機能を通じて明示的に保存するもの以外は、ユーザーコンテンツがキャッシュまたは保存されることはありません。

データ削除

SlideTouchを削除すると、すべてのアプリデータが一緒に削除されます。元のPDFファイルはSlideTouchによって変更されることなく、保存した場所にそのまま残ります。
