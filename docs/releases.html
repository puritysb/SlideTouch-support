Release Notes

Version history and changelog

v1.0 2026

Core Features

  • PDF loading and full-resolution page rendering
  • On-device OCR with Apple Vision (English, Korean, Japanese)
  • AI inpainting with LaMa model for background restoration
  • Non-destructive layer-based editing with unlimited undo/redo
  • Inline text editing with rich text support (fonts, sizes, colors, alignment)

Object Editing

  • Object selection and removal with SAM2 segmentation
  • Stroke-based mask generation for precise object isolation
  • Object move with automatic inpainting of the source area

Shape Drawing

  • Rectangle, ellipse, arrow, and freeform curve tools
  • Customizable stroke color, fill color, and line width

Export

  • PDF export with all edits composited
  • Image export (PNG, JPEG)
  • Video slideshow export
  • Animated GIF export

Localization

  • English, Korean, and Japanese interface
