Support

If you're experiencing issues with SlideTouch, please check below.

System Requirements

  • macOS 15.1+ (Sequoia or later)
  • Apple Silicon (M1, M2, M3, M4) — Intel Macs are not supported
  • 8GB+ RAM recommended

Frequently Asked Questions

Why does SlideTouch require Apple Silicon?

SlideTouch uses Core ML models that rely on the Neural Engine found in Apple Silicon chips. Intel Macs don't have this hardware, and CPU-only inference would be impractically slow.

Can I edit regular (text-based) PDFs?

SlideTouch is designed specifically for image-based PDFs where text exists as pixels, not as editable text objects. For text-based PDFs, standard PDF editors are more appropriate.

Will my original PDF be modified?

Never. SlideTouch follows a strict non-destructive editing principle. All edits are stored as overlay layers. When you export, a new file is created, leaving the original untouched.

What languages does OCR support?

Currently English, Korean, and Japanese. Language support depends on the Apple Vision framework's recognition capabilities.

Is an internet connection required?

No. All processing is done entirely on your device. SlideTouch has no network capability and never uploads your files anywhere.

Inpainting looks blurry or has artifacts

Try using a smaller edit region. The inpainting model works best on small to medium patches. For very large areas, you may need to edit in multiple smaller steps.

The app is using a lot of memory

SlideTouch loads ML models on demand and unloads them after 30 seconds of inactivity. For very large PDFs (100+ pages), closing pages you're not actively editing will reduce memory usage.

Report a Bug

The easiest way to report bugs is through GitHub Issues.

  • Issues: github.com/puritysb/SlideTouch-support/issues

What to Include in Bug Reports

  • App version (SlideTouch > About SlideTouch)
  • macOS version and chip model (e.g., macOS 15.2, M2 Pro)
  • Which feature was being used (OCR, inpainting, export, etc.)
  • Steps to reproduce the issue
  • Screenshots if possible

Privacy

SlideTouch processes everything on your device and has no network access. For details, see our Privacy Policy.
